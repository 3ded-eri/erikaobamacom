人生に迷ってしまった？振り出しに戻ったような気がする？
そんなことない。これは『強くてニューゲーム』。
そのまま進もう。道は続く。

テーマ： 人生に迷った時や、何かに挑戦したあと振り出しに戻ってしまったと感じた時のリフレーミングについて
課題： スタートラインに戻ってしまう感覚と、似たように思えて実は以前とは全く違う状況にいるという概念をどう視覚的に表現するか
解決方法： 当事者の錯覚とリフレーミングの効果を無限に伸びる階段で表現する
メモ： 伸びる階段の作り方、キャラクターの位置を変形するオブジェクト上に固定する方法

テーマの背景

人生、時には迷ってしまうことや、何かに挑戦したり乗り越えようとした結果振り出しに戻ってしまったと感じることもあります。
でも、どんなに状況が変わらないように見えても、そこにいる今の自分は以前と全く同じ人間ではないはずです。
なぜなら経験値を得て、知恵もついているのですから。

「戻ってきた」はずのスタートラインは実際には別の場所、強くてニューゲームのステージなのです。

課題と解決方法

課題

スタートラインに戻ってしまう感覚と、似たように思えて実は以前とは全く違う状況にいるという概念をどう視覚的に表現するか

解決方法

当事者の錯覚とリフレーミングの効果を無限に伸びる階段で表現する

メモ

伸びる階段の作り方

  • Array + Simple Deform + keyframes

キャラクターの位置を変形するオブジェクト上に固定する方法

  • Geometry Nodes + Copy Transform
